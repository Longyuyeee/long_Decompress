# 通用系统策划案模板

## 1. 系统概述 (System Overview)
- **设计目的**: [为什么要设计这个系统？它能解决什么问题？]
- **系统核心**: [一句话描述系统的逻辑闭环]

## 2. 系统功能 (Core Features)
- **功能 A**: [描述该功能具体的表现形式与交互]
- **功能 B**: [同上]

## 3. 逻辑流程图 (Flow Logic)
- **核心流程**: [描述系统启动、执行、反馈、结束的逻辑]
- **异常逻辑**: [如果失败了，系统如何处理？]

## 4. UI/交互建议 (Interaction Design)
- **主界面元素**: [需要显示哪些数据和按钮？]
- **交互反馈**: [玩家操作后，视觉/听觉上应该有什么反馈？]

## 5. 待解决问题/风险点 (Open Issues)
- **潜在风险**: [可能导致不平衡或体验不佳的点]
- **备选方案**: [若该方案行不通，备选策略是什么？]
