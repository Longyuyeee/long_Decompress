# UI 交互数据渲染需求表 (UI-UX Data Spec)

## 1. 界面元素映射 (Element Mapping)
| UI 组件 | 数据来源 (Data Source) | 显示逻辑 | 交互反馈 |
| :--- | :--- | :--- | :--- |
| 主头像 | `PlayerData.avatar_url` | 异步加载图片 | 点击弹出详细面板 |
| 生命条 | `CombatData.hp / CombatData.max_hp` | 线性填充 (Progress Bar) | 扣血时带有红色残影动效 |

## 2. 状态渲染字典 (State Rendering)
- **不可用状态**: [如: 按钮灰色禁用, 置顶锁图标]
- **极值状态**: [如: 血量低于 20% 时屏幕红边闪烁]

## 3. 多语言/排版边界
- **文本最大长度**: [如: 标题 12 字, 描述 100 字]
- **超长处理**: [如: 自动截断并显示 "..." 或 自动换行]
