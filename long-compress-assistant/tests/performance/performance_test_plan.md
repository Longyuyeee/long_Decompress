# 胧压缩·方便助手 - 性能测试计划

## 1. 测试目标
评估应用在不同场景下的性能表现，确保满足用户需求。

## 2. 测试范围

### 2.1 压缩性能测试
- 不同文件大小（1MB, 10MB, 100MB, 1GB）
- 不同压缩格式（ZIP, TAR, GZIP, BZIP2, XZ）
- 不同压缩级别（1-9）
- 带密码加密压缩

### 2.2 解压性能测试
- 不同压缩格式解压
- 带密码解压
- 批量文件解压
- 大文件解压

### 2.3 内存使用测试
- 压缩过程中的内存峰值
- 解压过程中的内存峰值
- 长时间运行的内存泄漏
- 多任务并发内存使用

### 2.4 并发处理测试
- 同时压缩多个文件
- 同时解压多个文件
- 混合操作并发
- 系统资源竞争

### 2.5 大文件处理测试
- 超大文件（>2GB）压缩
- 超大文件解压
- 磁盘I/O性能
- 进度反馈实时性

## 3. 测试工具

### 3.1 Rust性能测试
- **criterion**: 基准测试框架
- **pprof**: 性能分析
- **heaptrack**: 内存分析
- **flamegraph**: 火焰图生成

### 3.2 前端性能测试
- **Lighthouse**: 网页性能分析
- **Web Vitals**: 核心网页指标
- **Playwright**: 性能监控
- **Chrome DevTools**: 性能分析

### 3.3 系统监控
- **sysinfo**: 系统信息收集
- **psutil** (Python): 跨平台监控
- **perf** (Linux): 系统性能分析
- **Windows Performance Monitor**: Windows性能监控

## 4. 测试场景

### 4.1 场景1: 小文件快速压缩
- **文件大小**: 1-10MB
- **文件数量**: 1-10个
- **压缩格式**: ZIP
- **性能指标**:
  - 压缩时间 < 5秒
  - CPU使用率 < 50%
  - 内存使用 < 100MB

### 4.2 场景2: 大文件高效压缩
- **文件大小**: 100MB-1GB
- **文件数量**: 1-5个
- **压缩格式**: ZIP, TAR.GZ
- **性能指标**:
  - 压缩速度 > 10MB/秒
  - 内存使用 < 500MB
  - 进度更新频率 > 1次/秒

### 4.3 场景3: 批量文件处理
- **文件数量**: 50-100个
- **总大小**: 500MB-1GB
- **操作**: 批量压缩/解压
- **性能指标**:
  - 并发处理能力
  - 任务调度效率
  - 错误恢复能力

### 4.4 场景4: 加密压缩性能
- **文件大小**: 10-100MB
- **加密算法**: AES-256
- **性能指标**:
  - 加密开销 < 30%
  - 内存使用增加 < 50%
  - 解密验证时间

### 4.5 场景5: 极限压力测试
- **文件大小**: 最大支持大小
- **并发操作**: 最大并发数
- **持续时间**: 长时间运行
- **性能指标**:
  - 稳定性
  - 资源泄漏
  - 错误处理

## 5. 性能基准

### 5.1 压缩性能基准
| 文件大小 | 压缩格式 | 目标时间 | 内存上限 | CPU上限 |
|---------|---------|---------|---------|--------|
| 10MB    | ZIP     | 3秒     | 50MB    | 30%    |
| 100MB   | ZIP     | 15秒    | 200MB   | 50%    |
| 1GB     | ZIP     | 90秒    | 500MB   | 70%    |
| 10MB    | TAR.GZ  | 5秒     | 30MB    | 40%    |
| 100MB   | TAR.GZ  | 25秒    | 150MB   | 60%    |

### 5.2 解压性能基准
| 文件大小 | 压缩格式 | 目标时间 | 内存上限 |
|---------|---------|---------|---------|
| 10MB    | ZIP     | 1秒     | 30MB    |
| 100MB   | ZIP     | 5秒     | 100MB   |
| 1GB     | ZIP     | 30秒    | 300MB   |

### 5.3 内存使用基准
| 操作类型 | 文件大小 | 允许内存峰值 | 允许内存泄漏 |
|---------|---------|-------------|------------|
| 压缩     | 100MB   | 200MB       | < 1MB/小时 |
| 解压     | 100MB   | 150MB       | < 1MB/小时 |
| 批量操作 | 1GB     | 500MB       | < 5MB/小时 |
