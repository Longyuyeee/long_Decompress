# 胧解压·方便助手 - UI/UX 设计文档

## 1. 设计概述

### 1.1 产品定位
"胧解压·方便助手"是一款智能解压工具，支持多种压缩格式，具有密码破解功能，提供批量操作和详细的解压报告。

### 1.2 设计原则
- **简洁直观**：界面清晰，操作流程简单
- **高效实用**：减少用户操作步骤，提高解压效率
- **信息透明**：实时显示进度和状态，让用户了解操作过程
- **灵活配置**：提供丰富的设置选项，满足不同用户需求
- **可访问性**：支持多语言，考虑视觉障碍用户需求

### 2.2 拖放支持设计

**拖放区域特性：**
- 视觉反馈：文件拖入时区域高亮显示
- 文件验证：实时检查文件格式，无效文件显示警告
- 批量支持：支持同时拖放多个文件
- 文件夹支持：支持拖放整个文件夹（自动扫描压缩文件）

**拖放交互流程：**
1. 用户拖放文件到指定区域
2. 区域视觉反馈（颜色变化、边框动画）
3. 文件验证和添加
4. 显示添加成功的文件列表

### 2.3 进度显示设计

**多层级进度显示：**
1. **总体进度**：所有文件的处理进度
2. **单个文件进度**：当前处理文件的解压进度
3. **密码尝试进度**：密码破解的尝试进度
4. **实时状态**：当前操作状态（解压中、密码尝试中、完成等）

**进度可视化：**
- 进度条：使用不同颜色区分状态（蓝色-进行中，绿色-成功，红色-失败）
- 百分比显示：精确显示进度百分比
- 时间估计：根据当前速度估算剩余时间
- 速度显示：实时显示处理速度（MB/s）

## 3. CLI 版本设计

### 3.1 命令行界面设计

**基本命令结构：**
```bash
# 基本用法
long-extract [选项] <文件...>

# 示例
long-extract archive.zip
long-extract *.zip *.rar
long-extract -o ./output archive.zip
long-extract -p password.txt archive.zip
```

**命令选项设计：**
```
用法: long-extract [选项] <文件...>

解压选项:
  -o, --output DIR       指定输出目录 (默认: 当前目录)
  -d, --directory        为每个压缩文件创建子目录
  -f, --force            覆盖已存在的文件
  -k, --keep             保留压缩文件

密码选项:
  -p, --password FILE    使用密码本文件
  -P, --password TEXT    直接指定密码
  -b, --brute-force      启用暴力破解模式
  -t, --try-count NUM    最大尝试次数 (默认: 1000)

性能选项:
  -j, --jobs NUM         并发任务数 (默认: 2)
  -c, --cpu NUM          CPU使用限制百分比
  -m, --memory NUM       内存限制 (MB)

输出选项:
  -q, --quiet            安静模式，只显示错误
  -v, --verbose          详细输出模式
  --progress             显示进度条
  --json                 输出JSON格式结果

其他选项:
  -h, --help             显示帮助信息
  -V, --version          显示版本信息
  --config FILE          使用配置文件
```
