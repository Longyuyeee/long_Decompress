# 胧压缩·方便助手 - 当前开发现状文档 (v2.7)

## 1. 项目愿景与设计哲学
*   **引擎强大，界面极简**：基于 Rust 的高性能核心，通过上下文感知的前端界面隐藏复杂性。
*   **Aero Harmony (大一统)**：采用“现代化、实色卡片、强动效、苹果风”的视觉规范。
*   **极致小工具化**：针对小窗口（720x540）深度优化，提供类专业工具的局部滚动与高频交互体验。

## 2. 已完成的关键里程碑
*   **全站 UI 重构 (Milestone 2.5)**：完成全站语义化 Tailwind 映射，解决所有亮暗色冲突。
*   **主题引擎 2.0**：支持 5 种核心模式（Light, Dark, Cyber, Twilight, Paper）及 13 种强调色。
*   **设置中心功能优化 (Milestone 2.6)**：
    *   移除冗余栏目，新增暴力破解引擎管理（外部 TXT 导入）。
    *   集成 Windows 注册表自启动控制。
*   **交互体验极致优化 (Milestone 2.7)**：
    *   **区域滚动标准**：全站实现“固定头部+局部滚动”，彻底解决长列表导致的整体位移。
    *   **Vault 3.0**：自适应 Excel 表格布局，自研 Aero Tooltip 悬浮窗，统计数据右下角悬浮化。
    *   **弹窗瘦身**：激进的响应式弹窗宽度，极简三属性表单录入（补全后端隐式载荷）。

## 3. 后续演进计划
*   **DLong 生态**：实现物理文件的导入导出与安全分发。
*   **暴力破解引擎后端对接**：实现具体的从 TXT 密码本中读取并尝试解压的任务流。
*   **右键菜单集成**：对接系统原生右键菜单（Tauri 侧）。
