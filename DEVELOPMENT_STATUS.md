# 胧压缩·方便助手 - 当前开发现状文档 (v2.1)

## 1. 项目愿景与设计哲学
*   **引擎强大，界面极简**：基于 Rust 的高性能核心，通过上下文感知的前端界面隐藏复杂性。
*   **交互美学**：采用苹果风（Apple Style）、毛玻璃质感（Glassmorphism）、高频动效（GSAP）。
*   **任务中心**：一切操作围绕文件任务展开，摆脱传统管理软件的层级负荷。

## 2. 已完成的核心架构 (Milestone 1-5)

### 后端 (Rust/Tauri)
*   **生命线集成**：在 `main.rs` 中完整注册了压缩、解压、密码管理、任务队列等 20+ 核心命令。
*   **实时通信**：重构了 `CompressionService`，支持通过 `Window.emit` 推送实时 IO 日志流、百分比进度和冲突事件。
*   **全格式驱动**：扩展支持 `ZIP, 7z, RAR, TAR, GZ, BZ2, XZ` 及其组合格式（如 `.tar.gz`）。
*   **数据安全**：集成了 SQLite 数据库，支持凭证的加密存储与分类。

### 前端 (Vue 3/TS/Vite)
*   **解压工作区**：
    *   **AeroTable**：具备格式感应徽章、局部配置高亮、平滑展开详情页。
    *   **五级密码阶梯**：逻辑为“手动输入 -> 高频建议 -> 遍历密码本 -> 外部字典 -> 穷举”。
    *   **冲突处理**：实现了单行原地决策与全局对比弹窗。
*   **压缩工作区**：
    *   **磁吸打组 (Magnetic Stacks)**：支持多选文件平滑聚合，实现组级配置继承。
*   **密码保险箱**：
    *   **吸入特效**：拖入 `.dlong` 或 `.txt` 时触发波纹扩散与文件“飞入”动画。
    *   **极简管理**：移除了主密码锁定（按需），增加了毛玻璃新增凭证弹窗。
*   **系统全局组件**：
    *   **灵动通知球**：悬浮于顶部的流体进度反馈。
    *   **系统负载计**：底部的 CPU/内存监控胶囊，支持算力手动调度。

## 3. 环境与工程化修复
*   **环境净化**：彻底清理了 `src-tauri` 目录下的不相关文件夹（博客残留及技能元数据）。
*   **端口锁定**：将开发端口强制统一为 `1420`，解决了与其他项目的端口冲突。
*   **编译对齐**：修复了 50+ 个由于接口升级导致的 Rust 编译错误及 Vue 运行时白屏问题。
