# 胧压缩·方便助手 - 动画和交互效果设计

## 1. 动画设计原则

### 1.1 苹果风格动画原则
- **自然流畅**：使用缓动函数模拟物理运动
- **目的明确**：每个动画都有明确的功能目的
- **性能优先**：确保动画流畅不卡顿
- **一致性**：相同交互使用相同动画模式
- **可访问性**：提供减少动画的选项

### 1.2 动画性能优化
```css
/* 硬件加速优化 */
.animated-element {
  transform: translateZ(0);
  will-change: transform, opacity;
  backface-visibility: hidden;
}

/* 减少重绘 */
.animate-transform {
  /* 使用transform而不是top/left */
  transform: translateX(100px);
}

.animate-opacity {
  /* 使用opacity而不是display */
  opacity: 0;
  transition: opacity 0.3s ease;
}
```

## 2. 基础动画系统

### 2.1 动画持续时间
```css
/* 标准动画持续时间 */
:root {
  --duration-instant: 0ms;     /* 即时反馈 */
  --duration-fast: 100ms;      /* 微交互 */
  --duration-normal: 300ms;    /* 标准过渡 */
  --duration-slow: 500ms;      /* 强调动画 */
  --duration-slower: 700ms;    /* 复杂动画 */
}
```

### 2.2 缓动函数
```css
/* 标准缓动函数 */
:root {
  /* 线性 */
  --ease-linear: cubic-bezier(0, 0, 1, 1);

  /* 标准缓入 */
  --ease-in: cubic-bezier(0.42, 0, 1, 1);

  /* 标准缓出 */
  --ease-out: cubic-bezier(0, 0, 0.58, 1);

  /* 标准缓入缓出 */
  --ease-in-out: cubic-bezier(0.42, 0, 0.58, 1);

  /* 弹性动画 */
  --ease-elastic: cubic-bezier(0.68, -0.55, 0.27, 1.55);

  /* 反弹动画 */
  --ease-bounce: cubic-bezier(0.68, -0.55, 0.27, 1.55);
}
```

## 13. 动画实现指南

### 13.1 Vue3动画实现
```vue
<template>
  <!-- 过渡动画 -->
  <transition
    name="fade"
    mode="out-in"
    @before-enter="beforeEnter"
    @enter="enter"
    @leave="leave"
  >
    <component :is="currentComponent" />
  </transition>

  <!-- 列表动画 -->
  <transition-group
    name="list"
    tag="div"
    class="file-list"
  >
    <div
      v-for="file in files"
      :key="file.id"
      class="file-item"
    >
      {{ file.name }}
    </div>
  </transition-group>
</template>

<script setup>
import { gsap } from 'gsap'

const beforeEnter = (el) => {
  el.style.opacity = 0
  el.style.transform = 'translateY(20px)'
}

const enter = (el, done) => {
  gsap.to(el, {
    opacity: 1,
    y: 0,
    duration: 0.5,
    ease: 'power2.out',
    onComplete: done
  })
}

const leave = (el, done) => {
  gsap.to(el, {
    opacity: 0,
    y: -20,
    duration: 0.3,
    ease: 'power2.in',
    onComplete: done
  })
}
</script>

<style scoped>
/* Vue过渡类 */
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease, transform 0.3s ease;
}

.fade-enter-from,
.fade-leave-to {
  opacity: 0;
  transform: translateY(20px);
}

/* 列表过渡 */
.list-move {
  transition: transform 0.3s ease;
}

.list-enter-active,
.list-leave-active {
  transition: all 0.3s ease;
}

.list-enter-from,
.list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}
</style>
```

// 测量动画性能
const measureAnimationPerformance = () => {
  const startTime = performance.now()

  // 触发动画
  element.classList.add('animated')

  // 监听动画结束
  element.addEventListener('animationend', () => {
    const endTime = performance.now()
    const duration = endTime - startTime

    console.log(`动画耗时: ${duration.toFixed(2)}ms`)

    // 检查丢帧
    if (duration > 16.67) { // 超过60fps的一帧时间

    }
  })
}

const checkHardwareAcceleration = () => {
