## 设计系统核心特性

### 颜色系统
- **主色板**: 蓝色系（#0ea5e9）
- **辅助色板**: 灰色系（#64748b）
- **功能色**: 成功绿、警告黄、错误红、信息蓝
- **毛玻璃效果**: 明暗模式支持

### 排版系统
- **主要字体**: Inter
- **等宽字体**: JetBrains Mono
- **字号阶梯**: 从12px到48px的完整阶梯
- **字重系统**: 400、500、600、700

### 间距系统
- **基础单位**: 4px (0.25rem)
- **间距阶梯**: xs(4px)、sm(8px)、md(16px)、lg(24px)、xl(32px)、2xl(48px)

### 圆角系统
- **圆角阶梯**: none、sm(2px)、md(6px)、lg(8px)、xl(12px)、2xl(16px)、full

### 阴影系统
- **标准阴影**: sm、md、lg、xl、2xl
- **玻璃阴影**: glass、glass-hover（双层阴影效果）

### 动画系统
- **持续时间**: 快速(150ms)、常规(300ms)、慢速(500ms)
- **缓动函数**: ease-in、ease-out、ease-in-out
- **动画类型**: 淡入淡出、滑动、缩放、旋转、脉动、弹跳

## 使用方法

### 1. 使用CSS变量

```css
.custom-element {
  color: var(--color-primary);
  padding: var(--spacing-4);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-glass);
}
```

### 2. 使用Tailwind类

```html
<div class="bg-primary text-white p-4 rounded-lg shadow-glass">
  使用设计系统的元素
</div>
```

### 3. 使用TypeScript设计令牌

```typescript
import { designTokens } from '@/tokens'

// 使用颜色
const primaryColor = designTokens.colors.primary.DEFAULT

// 使用间距
const spacing = designTokens.spacing[4] // "1rem"

// 使用圆角
const borderRadius = designTokens.borderRadius.lg // "0.5rem"
```

### 4. 使用UI组件

```vue
<template>
  <GlassCard hoverable class="p-6">
    <h3 class="text-xl font-semibold">标题</h3>
    <p class="text-gray-600 mt-2">内容</p>
    <GlassButton variant="primary" class="mt-4">
      操作按钮
    </GlassButton>
  </GlassCard>
</template>
```
