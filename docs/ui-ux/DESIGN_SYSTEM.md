# 设计系统 - 胧压缩·方便助手

## 概述

本设计系统为"胧压缩·方便助手"提供一致的设计语言和UI组件规范，采用苹果毛玻璃设计风格，注重美观性、可用性和性能。

## 设计原则

### 1. 清晰明了 (Clarity)
- 界面元素层次分明
- 信息架构清晰
- 操作流程直观

### 2. 高效实用 (Efficiency)
- 减少不必要的操作步骤
- 提供快捷操作
- 优化性能表现

### 3. 美观优雅 (Elegance)
- 苹果毛玻璃设计风格
- 流畅的动画过渡
- 精致的细节处理

### 4. 安全可靠 (Security)
- 明确的安全提示
- 可靠的文件处理
- 隐私保护设计

## 排版系统

### 字体家族

- **主要字体**: Inter
  - 常规文本: `font-normal`
  - 中等文本: `font-medium`
  - 粗体文本: `font-semibold`, `font-bold`

- **等宽字体**: JetBrains Mono
  - 代码片段
  - 文件路径
  - 技术信息

### 字号阶梯

| 层级 | 字号 | 行高 | 使用场景 |
|------|------|------|----------|
| H1 | `2.5rem` (40px) | 1.2 | 页面主标题 |
| H2 | `2rem` (32px) | 1.25 | 区块标题 |
| H3 | `1.5rem` (24px) | 1.3 | 卡片标题 |
| H4 | `1.25rem` (20px) | 1.35 | 小节标题 |
| Body Large | `1.125rem` (18px) | 1.5 | 大段正文 |
| Body | `1rem` (16px) | 1.5 | 常规正文 |
| Body Small | `0.875rem` (14px) | 1.5 | 辅助文本 |
| Caption | `0.75rem` (12px) | 1.4 | 说明文字 |

### 字重使用

- **400 (Normal)**: 常规正文
- **500 (Medium)**: 强调文本、按钮文字
- **600 (Semibold)**: 标题、重要信息
- **700 (Bold)**: 主标题、特别强调

## 阴影系统

### 玻璃效果阴影
```css
box-shadow:
  0 8px 32px 0 rgba(31, 38, 135, 0.37),
  inset 0 1px 0 0 rgba(255, 255, 255, 0.1);
```

### 交互阴影
- 悬停: `shadow-lg` → `shadow-xl`
- 激活: `shadow-md` → `shadow-sm`
- 聚焦: `ring-2 ring-primary/50`

## 动画系统

### 持续时间
- 快速: `150ms`
- 常规: `300ms`
- 慢速: `500ms`

### 缓动函数
- 进入: `ease-out`
- 退出: `ease-in`
- 交互: `ease-in-out`

### 动画类型
1. **淡入淡出** (`fade-in`, `fade-out`)
2. **滑动** (`slide-up`, `slide-down`, `slide-left`, `slide-right`)
3. **缩放** (`scale-up`, `scale-down`)
4. **旋转** (`spin`, `pulse`)

## 组件规范

### 1. 玻璃卡片 (GlassCard)
- 背景: 毛玻璃效果
- 边框: 半透明边框
- 阴影: 双层阴影
- 圆角: `rounded-xl` (12px)
- 内边距: `p-6` (24px)

### 2. 玻璃按钮 (GlassButton)
- 尺寸: `sm` `md` `lg` `xl`
- 变体: `default` `primary` `secondary` `danger`
- 状态: `normal` `hover` `active` `disabled` `loading`
- 交互: 悬停缩放、点击反馈

### 3. 文件拖放区 (FileDropzone)
- 状态: `normal` `dragging` `error`
- 反馈: 视觉状态变化、进度提示
- 验证: 文件类型、大小限制

### 4. 进度条 (ProgressBar)
- 变体: `primary` `secondary` `success` `warning` `danger` `info`
- 样式: `solid` `striped` `animated`
- 状态: `determinate` `indeterminate`

## 布局系统

### 栅格系统
- 使用Tailwind的12列栅格
- 响应式断点: `sm` `md` `lg` `xl` `2xl`
- 容器最大宽度: `max-w-7xl`

### 页面结构
1. **顶部导航**: 应用标识、主要导航
2. **主要内容区**: 页面核心内容
3. **侧边栏**: 辅助信息、快捷操作
4. **底部信息**: 版权信息、次要链接

## 响应式设计

### 断点定义
- `sm`: 640px (移动设备)
- `md`: 768px (平板)
- `lg`: 1024px (桌面)
- `xl`: 1280px (大桌面)
- `2xl`: 1536px (超大桌面)

### 响应式策略
1. **移动优先**: 从小屏幕开始设计
2. **渐进增强**: 大屏幕增加功能
3. **内容优先**: 确保核心内容可访问

## 无障碍设计

### 键盘导航
- 所有交互元素支持键盘访问
- 清晰的焦点指示器
- 合理的Tab顺序

### 屏幕阅读器
- 语义化的HTML结构
- 适当的ARIA属性
- 有意义的替代文本

### 颜色对比
- 正文文本: 至少4.5:1
- 大号文本: 至少3:1
- 交互元素: 至少3:1

### 图标尺寸

```javascript
  }

```
