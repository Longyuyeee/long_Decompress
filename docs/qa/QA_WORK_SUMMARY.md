## 测试框架特点

### 架构设计
```
测试金字塔:
E2E测试 (10%) - Playwright（多浏览器）
     ↑
集成测试 (20%) - Vitest + Tauri模拟
     ↑
单元测试 (70%) - Vitest + @vue/test-utils
```

### 性能标准
| 操作类型 | 文件大小 | 最大时间 | 最大内存 |
|---------|---------|---------|---------|
| 压缩 | 10MB | 3秒 | 50MB |
| 解压 | 10MB | 2秒 | 30MB |
| 批量处理 | 100MB | 15秒 | 200MB |

### 质量标准
- 代码覆盖率目标: 70%+
- 所有测试必须通过CI/CD
- 安全扫描无高危漏洞
