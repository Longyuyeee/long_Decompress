## 密码强度算法设计

### 评分系统（0-100分）
1. **基础分**：50分
2. **长度加分**：最多20分（8-128字符）
3. **字符类型加分**：最多25分
   - 小写字母：5分
   - 大写字母：5分
   - 数字：5分
   - 符号：10分
4. **熵值加分**：最多20分
   - 20-40 bits：5分
   - 40-60 bits：10分
   - 60+ bits：20分
5. **问题扣分**：根据严重程度
   - 低：2分
   - 中：5分
   - 高：10分
   - 严重：20分

### 强度等级分类
- **0-20分**：非常弱
- **21-40分**：弱
- **41-60分**：中等
- **61-80分**：强
- **81-100分**：非常强

### 问题检测类型
1. **长度问题**：太短、太长
2. **字符类型缺失**：缺少大小写、数字、符号
3. **常见模式**：常见密码、字典单词
4. **序列模式**：连续字符、重复字符
5. **键盘模式**：键盘连续按键
6. **个人信息**：可能包含个人信息
7. **日期模式**：日期格式
