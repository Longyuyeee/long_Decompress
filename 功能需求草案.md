# 胧压缩·方便助手 - 功能需求草案

## 1. 核心解压功能

### 1.1 支持的压缩格式
- **基础格式**：ZIP、RAR、7Z、TAR、GZ、BZ2、XZ
- **加密格式**：支持AES加密的ZIP、RAR5、7Z加密
- **其他格式**：ISO、ARJ、CAB、LZH、Z、UUE、XXE等
- **分卷压缩**：支持分卷文件自动识别和合并

### 1.2 批量处理
- **文件队列**：支持拖放多个文件或文件夹
- **处理模式**：并行处理（多线程）、串行处理（顺序）
- **智能识别**：自动识别压缩文件类型和加密状态
- **进度监控**：实时显示每个文件的处理进度

### 1.3 密码尝试
- **密码本管理**：内置密码本存储和管理
- **尝试策略**：顺序尝试、常用密码优先、智能跳过
- **成功记录**：自动记录成功密码到历史
- **中断恢复**：支持暂停和继续，记录尝试进度

### 1.4 暴力破解（高级功能）
- **字符集配置**：数字、小写字母、大写字母、特殊符号
- **长度范围**：最小到最大密码长度设置
- **模式攻击**：常见密码模式、字典攻击、组合攻击
- **性能优化**：多线程暴力破解、GPU加速（可选）
- **进度估计**：基于尝试速度估算剩余时间
- **资源限制**：CPU/内存使用限制、温度监控
- **法律声明**：明确使用范围和责任声明

## 2. 核心压缩功能

### 2.1 支持的压缩格式
- **创建格式**：ZIP、7Z、TAR、GZ、BZ2、XZ
- **加密压缩**：支持密码保护的ZIP、7Z加密
- **压缩级别**：存储、最快、较快、标准、较好、最好
- **分卷压缩**：支持创建指定大小的分卷文件

### 2.2 批量压缩
- **文件队列**：支持批量添加文件和文件夹
- **压缩模式**：单独压缩、合并压缩、按文件夹压缩
- **智能配置**：根据文件类型自动选择最佳压缩方式
- **模板系统**：保存常用压缩配置为模板

### 2.3 高级压缩选项
- **加密设置**：使用密码本中的密码或新设密码
- **压缩算法**：选择不同的压缩算法（DEFLATE、LZMA、PPMd等）
- **文件过滤**：按类型、大小、日期过滤文件
- **元数据**：添加注释、保存文件属性

### 2.4 顺序执行
- **任务队列**：压缩和解压任务统一队列管理
- **优先级设置**：设置任务优先级
- **依赖关系**：设置任务间的依赖关系
- **资源控制**：控制CPU和内存使用限制

## 3. 密码本管理系统

### 2.1 密码存储
- **存储格式**：加密的SQLite数据库
- **分类系统**：按用途、来源、强度分类
- **标签管理**：支持多标签标记密码
- **导入导出**：支持文本、CSV、JSON格式

### 2.2 智能功能
- **密码生成**：基于规则的密码生成器
- **模式识别**：识别常见密码模式（生日、电话、简单组合）
- **去重合并**：自动识别和合并重复密码
- **统计分析**：密码强度分析、使用频率统计

### 2.3 安全保护
- **主密码**：使用主密码加密数据库
- **自动锁定**：闲置一段时间后自动锁定
- **审计日志**：记录密码访问和使用历史
